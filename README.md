## ✨ Overview

**Peter Answers** is a deceptive web application designed to prank your friends. To the victim, it looks like a mystical oracle capable of answering any question. In reality, the "oracle" is you, secretly typing the answer while pretending to type a formal petition.

## 🌙 Features

- **📺 Cinematic Experience**: Looping high-definition video background for a mystical atmosphere.
- **💎 Glassmorphic UI**: Sleek, modern interface using backdrop blurs and subtle white borders.
- **🤖 Gemini AI Integration**: If no secret answer is provided, the AI generates hilariously dramatic and cryptic responses through OpenRouter.
- **🤫 The "Trick"**: A stealthy input mechanism that allows the prankster to hide the answer within the petition field.
- **📱 Fully Responsive**: Optimized for desktop and mobile pranking sessions.

## 🗝️ The Secret Ritual

1. **The Petition**: In the "Petition" field, start by typing a period (`.`).
2. **The Reveal**: Whatever you type after the first period will be hidden from the screen; instead, the field will automatically fill with a predefined mystical phrase.
3. **Closing**: End your secret answer with another period (`.`) to return to normal typing mode.
4. **The Question**: Type any question and hit "Summon Answer". The oracle will speak the exact answer you secretly typed earlier.
5. **The Fallback**: If you don't use the secret ritual, Chirag (the AI) will deliver a theatrical 1-sentence response that will leave the seeker bewildered.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone [repository-url]
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env` file in the root directory and add your OpenRouter API Key:
   ```env
   VITE_OPENROUTER_API_KEY=your_api_key_here
   ```

4. **Run development server**:
   ```bash
   npm run dev
   ```
